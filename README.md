Medicare - Hospital Management System
This project is a simple Hospital Management REST API developed using Node.js and Express.

Features
Add patient details
View patient records
REST API implementation
JSON-based storage
Technologies Used
Node.js
Express.js
Thunder Client
GitHub
How to Run
Install dependencies npm install

Start server node index.js

Open browser or Thunder Client http://localhost:3000

API Endpoints
POST /patients - Add patient
GET /patients - View all patients
